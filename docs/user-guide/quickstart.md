# 快速开始

> 5 分钟上手 WubiTypeMaster

## 1. 启动应用

双击桌面图标或应用菜单中的 **WubiTypeMaster**。

> 首次启动会自动创建数据库并灌入五笔码表（约 8 万条，< 1 秒）。

## 2. 切换到五笔输入法

启动后，在系统托盘/输入法切换器中切换到**五笔 86 版输入法**。

## 3. 选择练习模式

首页提供 4 种模式入口：

| 模式                 | 适合场景               |
| -------------------- | ---------------------- |
| 📝 **文章跟打**      | 完整文章练习、综合提升 |
| 👾 **Word Invaders** | 趣味练习、字词反应     |
| 🫧 **Bubble**        | 轻松训练、词组练习     |
| 🎯 **KeyDrill**      | 突破弱键、专项提升     |

## 4. 开始练习

### 4.1 文章跟打

- 屏幕顶部显示目标文本
- 在输入框中用五笔逐字输入
- 错字会高亮红色，并弹出正确编码提示
- 完成时显示统计结果

### 4.2 Word Invaders

- 汉字从屏幕顶部下落
- 键入正确的字击毁它
- 漏过的字扣血

### 4.3 Bubble

- 词语装在气泡中上升
- 触顶前键入完成
- 连续完成获得连击加分

### 4.4 KeyDrill

- 系统根据你的弱键统计
- 重点出现含弱键的字词
- 高强度训练薄弱键位

## 5. 查看统计

进入 **统计** 页面查看：

- 历史记录
- WPM / 准确率趋势
- 26 键位热力图
- 错字本

## 6. 个性化设置

进入 **设置** 调整：

- 主题（Light / Dark）
- 强调色
- 虚拟键盘显隐
- 音效开关
- 自动更新源
