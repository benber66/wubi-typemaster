# 模式详解

> 各练习模式的详细说明

## 1. 文章跟打

### 用途
完整文章段落跟打，模拟真实工作场景，最能反映实际打字能力。

### 界面
```
┌────────────────────────────────────┐
│  📝 文章跟打  WPM: 65  准确率: 96% │
├────────────────────────────────────┤
│  中华人民共和国成立于1949年...     │ ← 目标文本
│  ████████████████░░░░░░░░░░░░░░  │ ← 进度条
├────────────────────────────────────┤
│  [输入框：用五笔输入]              │
│  中华人民共和国成立于...           │ ← 已输入
└────────────────────────────────────┘
```

### 计分
- **WPM** = 正确字数 ÷ 用时(分钟) × 5（汉字按 5 键/字计算）
- **准确率** = 正确字数 ÷ 总输入字数

### 错误处理
输入错误时：
- 当前字标红
- 弹出提示卡片，显示：
  - 正确汉字 + 红色边框
  - 五笔编码：`gjjg`（标注字根位置）
  - 常见词组：`中华人民共和国 → gjjg...`

## 2. Word Invaders（单词入侵者）

### 用途
趣味化的单字 / 词组练习，提高反应速度。

### 界面
```
        甲            我们
                          我们
              学
   生
                            我
              ┌──────────┐
              │  输入框  │
              └──────────┘
        血量: ♥♥♥  分数: 1250
```

### 规则
- 汉字从屏幕顶部以不同速度下落
- 在输入框键入下落中的字 → 击毁
- 漏过的字 → 扣 1 血
- 血量为 0 → 游戏结束
- 难度递增：随时间推移下落加速

### 计分
- 击毁单字：+10 分
- 击毁词组：+10 × 词组字数 分
- 连击：连续击毁无失误，分数 × 1.5 / × 2 / × 3

## 3. Bubble（气泡游戏）

### 用途
词组专项训练，节奏舒缓。

### 界面
```
    🫧(中国)              🫧(学习)
  🫧(我们)                   🫧(打字)
        🫧(速度)
   连击: x5  分数: 850
```

### 规则
- 词语装在气泡中从底部上升
- 在触顶前键入完成 → 气泡破裂
- 漏过 3 个 → 游戏结束

### 计分
- 完成：+20 × 词长
- 连击：x1.5 / x2 / x3 倍率

## 4. KeyDrill（弱键突破）

### 用途
针对个人薄弱键位的高强度训练。

### 算法
1. 读取 `key_stats` 表
2. 找出错误率最高的 3-5 个键位
3. 从五笔码表中筛选**含这些键位**的字词
4. 随机出题

### 界面
类似 Word Invaders，但出题严格基于弱键。

### 样本不足时
- 若历史练习 < 50 次，回退到**通用高频弱键**（基于五笔学习者共同难点）
- 提示用户："继续练习以解锁个性化弱键训练"

## 5. 模式对比

| 维度 | 文章跟打 | Word Invaders | Bubble | KeyDrill |
|---|---|---|---|---|
| 难度 | 中 | 中-高 | 中 | 中-高 |
| 强度 | 中 | 高 | 中 | 极高 |
| 趣味性 | 低 | 高 | 中 | 中 |
| 综合能力 | ★★★★★ | ★★★ | ★★ | ★★（针对） |
| 词组比重 | 高 | 中 | 高 | 中 |
| 单字比重 | 中 | 高 | 低 | 高 |
