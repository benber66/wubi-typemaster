# WubiTypeMaster 需求规格书

> 版本：v0.0.1（初始）
> 状态：已确认
> 最近更新：2026-06

## 1. 项目总览

### 1.1 项目名称
**WubiTypeMaster**（暂定，简称 WT 速练）

### 1.2 项目目标
开发一款跨平台桌面五笔字型打字练习软件，复刻经典 TypeMaster Typing Tutor 桌面版的核心体验，并针对五笔输入法的特点进行专项强化。

### 1.3 目标用户
**五笔 86 版进阶者**——已掌握五笔字根和基本拆字规则，需要提升打字速度和准确率的熟练用户。

### 1.4 平台
- 主目标：Windows、Linux
- 预留：macOS（暂不构建，但配置上保留可能性）

### 1.5 五笔版本支持
**仅支持 86 版五笔**。

## 2. 功能模块

### 2.1 练习模式（4 种）

#### 2.1.1 文章跟打模式
- 加载内置或导入的文本，逐字跟打
- 进度条 + 实时 WPM + 错字标记
- 完成统计（用时、平均速度、准确率）

#### 2.1.2 Word Invaders（单词入侵者）
- 汉字/词语从屏幕顶部下落
- 键入正确即击毁该对象
- 计分、难度递增

#### 2.1.3 Bubble（气泡游戏）
- 词语装在气泡中上升
- 触顶前键入完成使气泡破裂
- 连续完成获得连击加分

#### 2.1.4 KeyDrill（弱键突破）
- 基于个人弱键统计动态生成含弱键的字词
- 当样本不足时回退到通用高频字
- 针对薄弱键位进行高强度训练

### 2.2 输入流程

#### 2.2.1 主流程
- 用户在软件内文本框使用**系统五笔输入法**输入
- 软件捕获 IME 上屏的最终汉字，与目标字比对

#### 2.2.2 错误反馈
- 输入错误时，弹出错误提示卡片
- 提示内容包含：
  - 该汉字的标准五笔编码（4 字母）
  - 常见词组编码（如"我们"→"trwu"）

### 2.3 五笔码表

#### 2.3.1 数据范围
- 单字表：约 7000 常用汉字
- 词组表：常用 2-4 字词

#### 2.3.2 加载方式
开发期一次性整理为 JSON，编译/打包时灌入 SQLite 内置表，运行时查询。

### 2.4 统计与进度

#### 2.4.1 历史记录
每次练习记录：
- 日期时间
- 练习模式
- WPM（每分钟字数）
- 准确率
- 错字详情（错字、目标字、应输入编码）

#### 2.4.2 趋势图表
- 日/周/月维度的 WPM 趋势
- 日/周/月维度的准确率趋势

#### 2.4.3 键位热力图
- 26 个字母键位的错误率可视化
- 自动识别弱键
- 数据供 KeyDrill 模式使用

#### 2.4.4 错字本
- 累计常错汉字列表
- 可一键加入专项重练

### 2.5 词库与难度

- 一次性整理的常用字 + 常用词
- 抽取策略：**随机抽取，无固定分级**
- 不设"初级/中级/高级"等级

### 2.6 数据存储
- 本地 SQLite 数据库
- 存放位置：系统用户数据目录（按 OS 约定）
- 单用户使用，无云同步

### 2.7 用户界面

#### 2.7.1 主题
- 支持 Light / Dark 双主题
- 用户可自定义强调色

#### 2.7.2 虚拟键盘
- 五笔字根键盘可视化
- 输入时高亮按键、区分左右手
- **可隐藏**，默认对进阶者隐藏

#### 2.7.3 字体
- 打字显示使用等宽字体

#### 2.7.4 音效
- 可选，默认关闭
- 包含键入音、错误音、游戏背景音

## 3. 非功能性需求

### 3.1 性能
- 首屏加载 < 2s
- 输入反馈延迟 < 50ms
- 列表展示（历史记录、错字本）支持 1000+ 条无卡顿

### 3.2 兼容性
- Windows 10/11
- Ubuntu 22.04+（GNOME / KDE）
- 测试过的五笔输入法：极点五笔、海峰五笔、ibus-rime

### 3.3 隐私
- 所有数据本地存储
- 不上传任何用户数据
- 不包含任何第三方分析/追踪

## 4. 验收标准

### MVP（v0.1.0）
- 4 种练习模式均可运行
- 输入验证 + 错误提示五笔码
- 历史记录可查询
- 键位热力图可显示

### 正式版（v1.0.0）
- 跨平台打包产物可安装
- 自动更新可用
- 性能达标
- 文档完整
